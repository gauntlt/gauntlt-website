Gauntlt unites devs and ops and security work together to build rugged software. It is built to facilitate testing and communication between groups and create actionable tests that can be hooked into your continuous integration systems.

Features

  • Gauntlt attacks are written in easy-to-read language
  • Security tool support comes built-in
  • Integrated security tools include: sqlmap, nmap, sslyze and more
  • Easy to hook into your org's testing tools and processes

How do I get started?

$ gem install gauntlt
# download example attacks
# now you are ready to run the gauntlt
$ gauntlt
